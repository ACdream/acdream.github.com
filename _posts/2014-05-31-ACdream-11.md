A.梦

直接数学公式搞。

n个非负整数的和为m的解个数为C(n+m-1,n-1)

如果其中一个位置选择了i, 那么其余的就是C(n-1 + m-i – 1, n-2), 这就是这个位置i出现次数，  然后位置有n个，再乘以n.

	{
		if(u == n-1)
		{
			dp1[u] = dp2[u] = 0;
			return;
		}
		dp1[u] = dp2[u] = INF;

		{
